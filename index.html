Tentang Saya

Saya adalah mahasiswa yang antusias di bidang pengembangan web dan data, dengan minat mendalami Web Development (HTML, CSS, JavaScript, WordPress), Analisis Data menggunakan Pandas & Jupyter Notebook, serta penerapan DevOps (VPS EC2, CI/CD pipeline). Aktif mengikuti bootcamp dan kursus online untuk meningkatkan kompetensi teknis. Terbiasa bekerja mandiri maupun dalam tim, serta selalu belajar teknologi terbaru untuk mendukung perkembangan karier di dunia IT.

Projek Saya

Landing Page Portofolio

Landing page portofolio modern yang dibuat menggunakan HTML, CSS, dan JavaScript murni. Projek ini menampilkan desain responsif, navigasi interaktif, dan form kontak sederhana yang dapat digunakan sebagai template pribadi.

Analisis Dataset Sederhana

Menganalisis dataset sederhana menggunakan Python, Pandas, dan Jupyter Notebook. Projek ini bertujuan untuk memahami pola data, membuat visualisasi sederhana, dan menghasilkan insight dasar untuk mendukung pengambilan keputusan.

Deployment VPS dengan CI/CD

Mencoba proses deployment ke VPS EC2 dengan menerapkan pipeline CI/CD. Projek ini berfokus pada otomatisasi build, testing, dan deploy aplikasi ke server cloud agar pengembangan menjadi lebih efisien dan terstruktur.
